Web3 Starter project

Version 1.0.2

This project is web3 starter using angular 15 for internal usage. Using theme Finder and using finder's document
Author: a3r0nz

Features :
  • Connect metamask wallet

    This version only support metamask.

  • Display connected wallet

    The connected wallet will be shown instead of connect wallet button

  • Chain switcher

    You can config the support chains in utils/config.js file.

  • Sample page

    A sample to show your token balances using services.

  • Theme color changing

    You can config the colors like primary or secondary in assets/_variables.scss file. And if you want to custom the global scss you can use assets/_override.scss file
